# Data Cleaning Project

Этот проект демонстрирует процесс очистки "грязных" данных с использованием библиотек `pandas` и `numpy` в Python. Скрипт включает в себя несколько этапов очистки данных, таких как удаление строк с пропущенными значениями, заполнение пропусков, исправление ошибочных значений и удаление ненужных данных.

## Описание

В данном проекте создается "грязный" набор данных, содержащий пропущенные значения и ошибки. Скрипт выполняет следующие этапы очистки:

1. Удаление строк с пропущенными значениями в ключевых колонках.
2. Заполнение пропущенных значений средними значениями для числовых данных.
3. Исправление ошибочных данных (например, возраст и зарплата, которые отрицательные или слишком большие).
4. Удаление строк с ошибочными или слишком большими значениями в колонке зарплаты.

Результатом является очищенный набор данных, готовый для дальнейшего анализа.
